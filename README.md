## 簡介
氣泡排序（Bubble Sort）是一種簡單的排序演算法，<br>
其基本思想是反覆地比較相鄰的兩個元素，如果它們的順序錯誤就交換它們，直到整個數列排序完成。<br>
這個過程中，每一輪都會使得未排序部分的最大元素浮到數列的右側，就像氣泡從水底浮出一樣，因此得名氣泡排序。
